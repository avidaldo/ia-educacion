# NotebookLM

[NotebookLM](https://notebooklm.google.com/) é unha ferramenta de Google que funciona como un asistente de investigación persoal baseado en IA.

## Que o fai diferente?

A diferenza de ChatGPT ou Gemini, NotebookLM é **"source-grounded"** (fundamentado en fontes): só xera respostas baseándose nos documentos que ti sobes. 

## Funcionalidades principais

### 📚 Fontes soportadas
- PDFs e documentos de Google Docs
- URLs de páxinas web
- Vídeos de YouTube (transcrición automática)
- Arquivos de texto

### 🎧 Audio Overviews (Podcasts)
Unha das funcionalidades máis populares: NotebookLM pode xerar un **podcast conversacional** sobre os teus documentos.

- Dous "presentadores" discuten o contido de xeito natural
- Personalizable: duración, tono, enfoque
- Ideal para repasar contido "de camiño" a clase

### 🎬 Video Overviews
Similar aos podcasts pero en formato vídeo:
- Presentacións visuais xeradas automaticamente
- Dispoñible en máis de 80 idiomas
- Estilos visuais personalizables

### 📝 Study Guides e Quizzes
Xera automaticamente:
- **Guías de estudo** estruturadas
- **Preguntas de test** baseadas no contido
- **Flashcards** para memorización
- **Resumos** condensados

### 🗺️ Mind Maps
Mapas mentais interactivos para:
- Visualizar relacións entre conceptos
- Explorar conexións na túa investigación
- Sintetizar información de múltiples fontes

## Uso educativo

### Para estudantes
- Preparar exames xerando quizzes dos apuntamentos
- Escoitar resumos en podcast mentres estudan
- Organizar información de múltiples fontes
- Obter respostas verificables con citas

### Para docentes
- Crear materiais de estudo rapidamente
- Xerar preguntas de test sobre o temario
- Preparar resumos para os alumnos
- Crear podcasts educativos sen gravación

## Comparación con Proxectos de ChatGPT

| Característica | NotebookLM | ChatGPT Projects |
|---------------|------------|------------------|
| Baseado en fontes | ✅ Só os teus documentos | ❌ Tamén usa coñecemento xeral |
| Xera podcasts | ✅ Audio Overviews | ❌ |
| Citas automáticas | ✅ Con ligazóns | Parcial |
| Gratuíto | ✅ | Só con Plus |
| Ideal para | Investigación académica | Proxectos xerais |
