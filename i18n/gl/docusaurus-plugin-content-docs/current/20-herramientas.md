## Configuración de Chatbots

### Memoria

Os chatbots modernos poden lembrar información entre conversas:

| Chatbot | Característica | Descrición |
|---------|---------------|------------|
| **ChatGPT** | Memory | Lembra preferencias e datos entre chats. Activable/desactivable. |
| **Gemini** | Memory | Lembra contexto de conversas previas. Configurable. |
| **Claude** | Projects | Organiza conversas con contexto compartido. |

### Chats Temporais (Privacidade)

Para conversas que non queres que queden gardadas nin adestren o modelo:

- **ChatGPT**: *Temporary Chat* - bórrase automaticamente tras 30 días, non adestra o modelo
- **Gemini**: *Chat temporal* - bórrase tras 72 horas, non adestra o modelo, similar ao modo incógnito
- **Claude**: Opción de non compartir datos para adestramento

> [!TIP]
> Usa chats temporais cando trabales con información sensible ou confidencial.

### Canvas / Espazos de Traballo

Interfaces avanzadas para edición colaborativa coa IA:

**ChatGPT Canvas** ([documentación](https://openai.com/index/introducing-canvas/)):
- Panel de chat + editor integrado
- Edición inline: selecciona texto e pide cambios específicos
- Para código: depuración, comentarios, refactoring, execución de Python
- Para texto: axustar nivel de lectura, expandir/contraer, revisar gramática
- Control de versións: desfacer cambios, restaurar versións anteriores

**Gemini** tamén permite editar código e documentos de forma colaborativa, aínda que con menos funcionalidades que Canvas.
