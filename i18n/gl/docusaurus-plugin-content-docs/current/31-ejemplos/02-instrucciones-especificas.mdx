# Uso de exemplos de saída (one-shot / few-shot)

Dependendo da pregunta, pode ser útil proporcionar ao modelo exemplos de resposta.

- Zero-shot: Non se dan exemplos, só a instrución.
- One-shot: Inclúese un único exemplo/patrón para guiar.
- Few-shot: Ofrécense varios exemplos para maior claridade.
