# Using output examples (one-shot / few-shot)

Depending on the question, it may be helpful to provide the model with answer examples.

- Zero-shot: No examples are given, only the instruction.
- One-shot: A single example/pattern is included to guide you.
- Few-shot: Several examples are provided for clarity.
