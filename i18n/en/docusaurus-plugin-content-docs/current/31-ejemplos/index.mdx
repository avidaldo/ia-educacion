# Examples of Prompts

In this section you will find examples of “meta” prompts designed for **secondary and vocational training teachers**.

The idea is not to copy/paste *the content* that the AI ​​generates, but to **learn to ask for it** with context, restrictions and verification.

## Conclusion

These examples show how to improve results without fragile “tricks”:

- Start with the **teaching objective** and the **evidence** (what the students should produce).
- Control output with **formatting** (fixed headers, tables, limits).
- Reduce errors with **clarification questions** and a **critical review**.
- Add **privacy** restrictions: avoid personal data of the students.
