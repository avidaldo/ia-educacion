# Presentations with AI

Creating presentations is one of the most common tasks where AI can save significant time.

## Direct generation in chatbots

Any modern chatbot can create presentations in various formats:

### Markdown + Mermaid
```
Crea una presentación en Markdown sobre [tema].
Incluye diagramas en Mermaid donde sea apropiado.
Estructura: 10-15 diapositivas, estilo profesional.
```

The result can be:
- Convert to PowerPoint with tools like [Marp](https://marp.app/) or [Slidev](https://sli.dev/)
- Import into Google Slides
- Use directly on platforms that support Markdown

### HTML generation
Chatbots can also generate HTML presentations with [reveal.js](https://revealjs.com/), which you can then host or convert.

## Specialized tools

### [Gamma](https://gamma.app/) - Speed ​​and automation

**Ideal for**: quick presentations, drafts, web-style content

- Generate complete presentations from a prompt or text
- Flexible card system (more similar to a website than slides)
- AI generates content, design and images automatically
- Modern and attractive result with minimal effort
- Real-time collaboration

**Limitations**:
- Less control over pixel-perfect design
- Content may be generic (editing required)
- More limited asset library

### [Canva](https://www.canva.com/) - Control and versatility

**Ideal for**: branded presentations, detailed design, educational materials

- **Magic Design**: generate outline and fill slides from a prompt
- **Magic Write**: enhance and generate text
- **Magic Media**: generate images from text
- Library of +100 million photos, graphics, templates
- Full control over every design element
- [Canva for Education](https://www.canva.com/education/): free for teachers and students

**Limitations**:
- Requires more manual work than Gamma
- Full features require Pro subscription

### [Beautiful.AI](https://www.beautiful.ai/)
- Automatic layout that adapts to the content
- Professional templates
- Less known but very polished

### [slidesgpt](https://slidesgpt.com)
- Generate slides from text
- Export to PowerPoint directly

## Recommended Workflow for Teachers

1. **Quick Draft**: Use Gamma to generate a first version
2. **Refinement**: If you need more control, export and edit in Canva or PowerPoint
3. **For branded presentations**: Get started right in Canva with your Brand Kit

## Classroom integration

### Canva for Education
- Free for verified teachers and students
- Education-specific templates
- Real-time collaboration with students
- Integration with Google Classroom and LMS

### Collaborative generation with students
We can ask students to:
1. Generate a draft with AI
2. Identify what content is correct/incorrect
3. Improve and customize the design
4. Present explaining your changes
