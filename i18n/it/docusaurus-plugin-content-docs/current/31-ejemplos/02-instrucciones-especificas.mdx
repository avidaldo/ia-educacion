# Utilizzo di esempi di output (one-shot/pochi-shot)

A seconda della domanda, potrebbe essere utile fornire al modello esempi di risposte.

- Zero-shot: non vengono forniti esempi, solo le istruzioni.
- One-shot: è incluso un singolo esempio/modello per guidarti.
- Few-shot: vengono forniti diversi esempi per chiarezza.
