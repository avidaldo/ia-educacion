# Presentaciones con IA

La creación de presentaciones es una de las tareas más comunes donde la IA puede ahorrar tiempo significativo.

## Generación directa en chatbots

Cualquier chatbot moderno puede crear presentaciones en varios formatos:

### Markdown + Mermaid
```
Crea una presentación en Markdown sobre [tema].
Incluye diagramas en Mermaid donde sea apropiado.
Estructura: 10-15 diapositivas, estilo profesional.
```

El resultado se puede:
- Convertir a PowerPoint con herramientas como [Marp](https://marp.app/) o [Slidev](https://sli.dev/)
- Importar en Google Slides
- Usar directamente en plataformas que soporten Markdown

### Generación de HTML
Los chatbots también pueden generar presentaciones en HTML con [reveal.js](https://revealjs.com/), que luego puedes alojar o convertir.

## Herramientas especializadas

### [Gamma](https://gamma.app/) - Velocidad y automatización

**Ideal para**: presentaciones rápidas, borradores, contenido web-style

- Genera presentaciones completas desde un prompt o texto
- Sistema de tarjetas flexible (más similar a una web que a diapositivas)
- La IA genera contenido, diseño e imágenes automáticamente
- Resultado moderno y atractivo con mínimo esfuerzo
- Colaboración en tiempo real

**Limitaciones**:
- Menos control sobre el diseño pixel-perfect
- Contenido puede resultar genérico (requiere edición)
- Biblioteca de assets más limitada

### [Canva](https://www.canva.com/) - Control y versatilidad

**Ideal para**: presentaciones con marca, diseño detallado, material educativo

- **Magic Design**: genera outline y rellena diapositivas desde un prompt
- **Magic Write**: mejora y genera texto
- **Magic Media**: genera imágenes desde texto
- Biblioteca de +100 millones de fotos, gráficos, plantillas
- Control total sobre cada elemento de diseño
- [Canva for Education](https://www.canva.com/education/): gratis para docentes y estudiantes

**Limitaciones**:
- Requiere más trabajo manual que Gamma
- Funciones completas requieren suscripción Pro

### [Beautiful.AI](https://www.beautiful.ai/)
- Diseño automático que se adapta al contenido
- Plantillas profesionales
- Menos conocido pero muy pulido

### [slidesgpt](https://slidesgpt.com)
- Genera diapositivas desde texto
- Exporta a PowerPoint directamente

## Flujo de trabajo recomendado para docentes

1. **Borrador rápido**: Usa Gamma para generar una primera versión
2. **Refinamiento**: Si necesitas más control, exporta y edita en Canva o PowerPoint
3. **Para presentaciones con marca**: Empieza directamente en Canva con tu Brand Kit

## Integración en el aula

### Canva for Education
- Gratuito para docentes y estudiantes verificados
- Plantillas específicas para educación
- Colaboración en tiempo real con estudiantes
- Integración con Google Classroom y LMS

### Generación colaborativa con estudiantes
Podemos pedir a los estudiantes que:
1. Generen un borrador con IA
2. Identifiquen qué contenido es correcto/incorrecto
3. Mejoren y personalicen el diseño
4. Presenten explicando sus cambios
