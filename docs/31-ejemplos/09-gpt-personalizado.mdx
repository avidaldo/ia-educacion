# Asistentes IA Personalizados

Los asistentes personalizados te permiten guardar prompts específicos para tareas que repites frecuentemente. El usuario ve un chat normal, pero el prompt almacenado se añade automáticamente delante de cada mensaje.

## GPTs de ChatGPT

[GPTs personalizados](https://chatgpt.com/gpts) es una función de ChatGPT Plus que permite crear asistentes especializados.

### Cómo crear un GPT

1. Accede a [ChatGPT GPTs](https://chatgpt.com/gpts)
2. Haz clic en "Crear"
3. Define:
   - **Nombre** y descripción
   - **Instrucciones**: el prompt que se añadirá a cada conversación
   - **Conocimiento**: documentos que puede consultar
   - **Capacidades**: acceso a web, DALL-E, Code Interpreter...
4. Guarda y comparte (público o privado)

### GPTs educativos útiles

- [Mermaid Chart](https://chatgpt.com/g/g-1IRFKwq4G-mermaid-chart-diagrams-and-charts) - Genera diagramas
- [Sonia - Diagramas Educativos](https://chatgpt.com/g/g-7BAVTYCJr-sonia-diagramas-educativos-mermaid)
- [Generador de preguntas de test](https://chatgpt.com/g/g-67b721d49a0481918a78bc5104bd22ea-generador-de-preguntas-de-test) - Ver [Tests para Moodle](../51-tests-moodle.mdx)

:::warning[Error común]
En las instrucciones NO escribas "Crea un asistente que..." - escribe directamente el prompt que quieres que se use en cada conversación.

❌ Incorrecto: "Crea un generador de preguntas de test que..."
✅ Correcto: "Eres un experto en evaluación educativa. Cuando el usuario proporcione un tema..."
:::


## Gems de Gemini

[Gems](https://gemini.google.com/gems) es el equivalente de Google para Gemini. Disponible con Gemini Advanced (Google AI Pro).

### Cómo crear una Gem

1. Accede a [Gemini](https://gemini.google.com/)
2. Selecciona "Gems" en el menú lateral
3. Haz clic en "Nueva Gem"
4. Define nombre e instrucciones
5. Opcionalmente, puedes compartir la Gem

### Gems premade

Google ofrece Gems predefinidas:
- **Brainstormer**: para generar ideas
- **Career Guide**: asesor de carrera
- **Learning Coach**: tutor personalizado
- **Writing Editor**: corrector de estilo

También hay **Gems experimentales** de Google Labs disponibles para probar.


## Proyectos de Claude

Claude organiza el trabajo con *Projects*, que permiten:
- Compartir contexto entre múltiples conversaciones
- Adjuntar documentos de referencia
- Definir instrucciones personalizadas para el proyecto

Aunque no son exactamente "asistentes personalizados", cumplen una función similar para trabajo continuado sobre un tema.


## Comparativa

| | GPTs (ChatGPT) | Gems (Gemini) | Projects (Claude) |
|--|----------------|---------------|-------------------|
| **Disponible en** | Plus/Teams | Advanced | Todos (limitado en Free) |
| **Compartir públicamente** | ✅ | ✅ | ❌ |
| **Adjuntar documentos** | ✅ | Limitado | ✅ |
| **Acceso a web** | ✅ | ✅ | ❌ (solo docs adjuntos) |
| **Generar imágenes** | ✅ (DALL-E) | ✅ (Imagen) | ❌ |


## Casos de uso educativo

- **Corrector de trabajos**: instrucciones específicas sobre rúbrica y criterios
- **Tutor de asignatura**: prompt con temario y estilo de explicación
- **Generador de materiales**: crea ejercicios, tests, presentaciones
- **Traductor especializado**: traduce con terminología específica del área
- **Simulador de entrevistas**: practica respuestas para oposiciones o entrevistas
