# Pensamiento paso a paso

La estrategia de *Chain of Thought* (CoT) consiste en pedir al modelo explícitamente que realice una serie de pasos lógicos para resolver un problema. Sin embargo, actualmente casi todos los modelos de IA lo hacen de manera implícita (modelos con razonamiento).

>**Por qué funciona**:
>- Indica el formato de respuesta (explicación paso a paso).
>- Da un problema claro.
>- Obliga al modelo a mostrar su proceso de pensamiento.
