# Uso de ejemplos de respuesta

Dependiendo de la pregunta, puede ser útil proporcionar al modelo ejemplos de respuesta.

- Zero-shot: No se dan ejemplos, solo la instrucción.
- One-shot: Se incluye un único ejemplo para guiar.
- Few-shot: Se ofrecen varios ejemplos para mayor claridad.
